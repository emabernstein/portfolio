Usage

  • All your html pages should be in /public
  • Modify /tailwind.config.js as you wish
  • Install with yarn or npm install
  • Launch the dev server with yarn serve or npm run serve
  • Build for production with yarn build or npm run build
  • Build for development (all of Tailwind included) with yarn development or npm run development

    • Deploying to Netlify

  • Build Command: yarn build
  • Publish Directory: public/

# tailwind-template

Little template repository for basic tailwind stuff

## Running ⛹️‍♀️

should be able to

```sh
yarn # or npm i
yarn serve # or npm run serve
```

or just open `index.html` in your browser 😊

## Building 👷‍♀️

```sh
yarn build # or npm run build
```

just builds tailwind to not be huge!
